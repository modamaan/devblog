# DevBlog

DevBlog is a modern, personal blog platform built by developers, for developers. It's designed to be a clean, distraction-free environment for sharing technical deep-dives, coding tutorials, insights on neuroscience, finance concepts, and interdisciplinary thinking.

## üöÄ Features

- **Personal Blog Focus**: A curated space for deep technical insights and personal growth topics.
- **Topics Covered**:
  - Software Development & Programming
  - System Design & Technical Deep-Dives
  - Neuroscience & Cognitive Science
  - Finance & Wealth Concepts
  - Productivity & Personal Growth
- **Clean UI/UX**: Built with modern web standards to provide a visually pleasing and distraction-free reading experience.
- **Engagement**: While it is a personal publishing platform, readers can interact by sharing, liking, and commenting on articles.

## üèÉ Getting Started

First, run the development server:

```bash

# or

# or

# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can begin editing the application by modifying `app/page.tsx`. The page auto-updates as you edit the file.

## üìÅ Project Structure

- `/app`: Contains all Next.js app router pages, layouts, and API routes.
  - `/about`: About DevBlog page.
  - `/contact`: Contact information page.
  - `/terms`: Terms and Conditions.
  - `/privacy`: Privacy Policy.
- `/components`: Reusable UI components (Navbar, Footer, Buttons, etc.).
- `/lib`: Utility functions and helper methods.
- `/public`: Static assets (images, fonts, etc.).
